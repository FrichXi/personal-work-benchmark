# Tiny Web Task Prompt

Build a tiny static website from the local files in `input/`.

## Source Material

- `input/articles.json` contains two articles with `id`, `title`, `date`, `author`, and `raw_markdown`.
- `input/graph.json` contains demo nodes and links.

## Required Output

Write all output into the current working directory.

- `index.html`: home page with links to graph and articles.
- `graph/index.html`: page that renders or clearly lists the graph relationship.
- `articles/index.html`: list of all articles and links to details.
- `articles/001/index.html`: detail page for article `001`.
- `README.md`: local preview instructions.
- `VERIFICATION.md`: commands or checks used to verify the output.

Do not ask questions. Create the files directly.
